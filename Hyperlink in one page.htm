Ketahuilah Situs Anda secar Keseluruhan

Ketika sedang mengerjakan proyek web Anda, ketahuilah seluruh komponen yang ada, seperti berapa space yang dibutuhkan oleh seluruh file, berapa banyak file image, apakah ada link yang terputus, dan sebagainya. Untuk mengetahui hal ini Anda cukup memilih menu View*Report, dan mengklik Site Summary. Frontpage akan melakukan analisis, dan melaporkannya kapada anda. (Infokomputer Internet Vol. II no. 2)

Memformat teks Sebagai Subscript

Apakah Anda mengira bahwa pada halaman web Anda tdak bisa membuat subscript atau superscript untuk penulisan naskah ilmiah Anda? dengan Frontpage 2000, Anda mudah melakukannya. Seleksilah karakter atau teks yang hendak Anda buat sebagai subscript atau superscript, lalu pilih menu Format*Font. Tentukan format yang Anda inginkanpada tab Font lalu klik OK. Berhasil bukan? (Infokomputer Internet Vol. II no. 2)

Selalu Gunakan huruf Kecil

Pernahkah Anda meraa kebingungan setelah meng-upload web cintent dan menemukan bahwa tidak semua link-nya bisa berjalan, seperti ketika Anda mengujinya secaralicol? Hindari persaan frustasi ketika hal itu terjadi serta biaya pulsa yang dikeluarkan untuk membenahi secara remote. kuncinya, gunkan semua huruf kecil untuk nama file dan folber, baik path maupun link (kecuali ke situs eksternal milik orang lain). Anda tentu yahu bahwa server UNIX atau LINUX, huruf besar (uppercase) dan huruf kecil (lowercase) dibedakan. ANda juga tidak bisa menjamin bahwa Nada akan selau menggunakan server NT, karena mingkinsaja akan pindahatau ganti server.(Infokomputer Internet Vol. II no. 2)

Membuat Thumbnail

Salah satu cara meringankan "bobot" halaman web yang mengandung file image adalah dengan mengecilkan ukuran file image tersebut. tetapi adakalanya pengunjung web justru menginginkan image yang lebih jelas dan tentu saja berarti lebih besar. Untuk mengatasi dapat dilakukan denagn memasang file image yang berukuran kecil dihalaman web, dan jika pengunjung tersebut menghendaki image aslinya, pengunjung tersebut tinggal mengklik image kecil itu untuk menampilakan image yang asli. Image kecil ini disebut Thumbnail. Pada Frontpage, membuat thumbnail sama mudahnya dengan meresize dan resample image. klik pada image yang akan dibuat thumbnail, lalu klik icon Auto Thumbnail pada toolbar picture. Pada saat Anda menyimpan halan web yang mengandung file image yang dibuat thumbnail, Anda akan diminta untuk memberi nama file image kecil(thumbnail) yang tercipta. Anda juga dapat mengatur ukuran thumbnail ini beserta border dan level edge-nya melalui menu Tool*Page Option, lalu mengklik tab Auto Thumbnail. (Infokomputer Internet Vol. II no. 2)

Gambar Background Yang Tidak Bergeser

Apakah Anda ingin memsang gambar pemandangan atau foto sebagai latar belakang? Jika halaman web Anda panjang melebihi tinggi jendela browser, halaman web Anda bisa digulung ke bawah, dan gambar latar belakang akan di-tile untuk memenuhi halaman web. tentu saja gambar latar belakang ini akan tampak dobel bahkan gambar latar belakang bahakan bisa tampak cuma setengahnya. Bayangkan jika pemotongannya pas digambar hidung anda, pasti akan lucu tampaknya. Untuk menghindari gambar latar belakang yang bisa yang bisa digeser, aturlah gambar latar belakang tersebut sebagai. (Infokomputer Internet Vol. II no. 2)

Memasang Gambar Secara sepotong-Sepotong

Jika Anda memasang sebuah gambar ataou foto yang berukuran besar dihalaman web Anda, dapat dipastikan waktu loading halaman web tersebut akan lambat karena harus men-download file gambar yang berukuran besar jadi sebelum halaman web Anda tampil sempurna, pengunjung web Anda sudah bosan menunggu, lalu pindah ke alamat situs yang lain. Agar gambar yang besar dapat ditampilakan secara perlahan-lahan, pecahlah gambar tersebut menjadi beberapa bagian (misalnya dipotong menjadi 3x3) lalu digabungkan dengan tabel. Untuk memotong gambar ini dengan mudah, Anda dapat menggunakan software seperti Macromedia Firework. Dengan cara ini. selalu halaman Web Anda menjadi jauh ringan di-load, foto atau gambar Anda tersebut menjadi tidak mudah "dicolong" orang yang biasa menggunakan klik-kanan moouse dan memilih save As.... (Infokomputer Internet Vol. II no. 2)
